### 数组合并
>给与两个有序数组，一个正序一个倒序，如何最简合并

1. 允许调用API  
    思路：合并数组然后排序
```java
    public Integer[] combineArray(int [] a,int[] b){
        List<Integer> result = Stream.of(a,b).flatMapToInt(Arrays::stream).boxed().sorted().collect(Collectors.toList());
        return  result.toArray(new Integer[0]);
    }
```
